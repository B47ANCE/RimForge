- Detect changed mods using source fingerprints
- Rebuild only affected graph and report data

- Track Workshop update timestamps

- Unified DDS health report

- Audit, profile, dependency, and texture dashboards
- Progress, cancellation, and report navigation

- Stable command API and configuration schema
- Installer and signed release artifacts
- User documentation and troubleshooting guides
- Public compatibility-data contribution workflow
